Ce paramètre présentera lors du lancement d'un build une option pour sélectionner un tag Git (ou une référence), le choix sera prit en compte pour la paramétrisation du build.

Il faut savoir que git ne nous permet pas la récupération d'informations complémentaires (telles que l'auteur ou la date du commit) à partir d'une URL distance. Le plugin clone le projet lorsque l'environnement de travail est vide, cela peut prendre du temps en fonction de la connexion ou de la taille du projet.

Le plus souvent, le paramètres défini dans le champ "Nom" est utilisé pour spécifier la branche git à récupérer (checkout).
