Ten parametr prezentuje dane z repozytorium Git takie jak: dostępne gałęzie, tagi, rewizje (wraz z datą wprowadzenia i autorem).

Należy pamiętć, że Git nie udostępnia bezpośrednio takich danych jak autor/data wprowadzenia ze zdalnego repozytorium. Ta wtyczka pobiera te dane w tle. Dlatego prezentacja danych może potrwać chwilę jeżeli repozytorium jest duże lub łącze internetowe jest wolne.

Często wtyczka ta wykorzystywana jest do określenia z jakiej gałęzi ma być budowany projekt.
